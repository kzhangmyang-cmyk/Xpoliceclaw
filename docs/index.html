Safety Notes

What the website does not do

  • The hosted site does not scan or uninstall anything on the visitor machine.
  • Automatic handling is limited to clearly scoped user-level agent footprints, persistence entries, config paths, cache paths, and explicit binaries.
  • Blocked, terminate-only, and partial outcomes are expected safety states, not application failures.
  • Complex persistence, driver-level residency, browser store extensions, and broad user data trees remain outside automatic cleanup scope.
