Capability Scope

What the installed client does

Workbench Real scan and uninstall tasks

The desktop client opens the same workbench UI and runs background jobs locally.

Export JSON and DOCX reports

Structured evidence and human-readable reports remain available after each local run.

Recovery Persistent scan and uninstall history

The client restores handled state and residual review from the local runtime directory.

Guardrails Conservative deletion boundaries

Blocked and manual-review outcomes are normal when the target footprint is broad or ambiguous.

Safety Notes

What the website does not do

  • The hosted site does not scan or uninstall anything on the visitor machine.
  • Automatic handling is limited to clearly scoped user-level agent footprints, persistence entries, config paths, cache paths, and explicit binaries.
  • Blocked, terminate-only, and partial outcomes are expected safety states, not application failures.
  • Complex persistence, driver-level residency, browser store extensions, and broad user data trees remain outside automatic cleanup scope.
