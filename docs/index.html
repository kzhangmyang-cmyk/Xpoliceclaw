检测 真实本地后台扫描

客户端复用了 Python 扫描引擎，实际运行在本机，而不是网站服务器。

处置 保守型卸载流程

自动处置只覆盖边界清晰的用户级足迹，并把残留项保留下来供后续复核。

审计 报告和历史都留在本地

JSON、DOCX、扫描历史和卸载历史会保存在桌面客户端的运行时目录下。
