交付链路

官网、发布与本地工作台的边界

  • 1. GitHub Pages 托管下载控制台

    xpoliceclaw.com 只展示下载入口、平台状态、能力边界与架构说明。

  • 2. GitHub Releases 分发桌面安装包

    Windows 与 macOS 安装包都由公开 Release 提供，页面按钮和稳定直链都指向这里。

  • 3. 本地客户端执行真实扫描与处置

    安装后，扫描、证据、卸载、历史恢复与残留复核都在用户自己的设备上完成。

Safety Notes

能力边界与风险声明

  • 只分发客户端，不远程扫描

    官网不会扫描访问者设备，也不会在浏览器里执行卸载动作。

  • 自动处置是保守型的

    Blocked、Partial、Manual Review 都是正常结果，不代表系统异常。

  • 只聚焦 AI agents 与本地执行风险

    产品不扩展为通用杀毒或全盘系统防护工具。

  • 报告、历史与证据保留在本地

    JSON、DOCX、scan history 与 uninstall history 都保存在客户端运行时目录。
